# Roadmap

> *The forest knows where it is going. This is how.*

The Spectorium roadmap tracks where the platform is headed — in versioned milestones, organized by horizon. Near is specific. Far is directional.

---

## Current — v0.1.0 · *First Light through The Forest*

**Status:** Active development

What exists now:
- [x] The Spectorium platform — root structure, Charter, standards
- [x] 31-tree arboreal taxonomy (Oak Pinnacle + 3 tiers of 10)
- [x] Spector's Grove — all files, Turn 0 through Turn 4
- [x] GLOSSARY.md — all terms defined
- [x] ROADMAP.md — this file
- [x] SUDOKU_ENGINE.md — concept bridge
- [x] Grove templates — copy-to-plant scaffolding
- [x] Terminal scripts — `new-grove.sh`, `grove-status.sh`
- [x] GitHub issue templates — Grove request, turn proposal, tree addition
- [x] GitHub Actions — Grove structure validation on PR

## Horizon — v0.3.0 · *Roots and Signals*

**Theme:** The platform becomes operational, not just documented.

- [ ] Machine-readable Grove manifests (YAML alongside MANIFEST.md)
- [ ] `grove-validate` script — local pre-commit Grove structure check
- [ ] `grove-link` script — establish connections between two Groves in one command
- [ ] Telegard protocol: formalized signal/dispatch schema
- [ ] Sudoku Engine integration — prime/composite device classification in GROVES.md
- [ ] First Tier 2 (Digital Grove) device registered
- [ ] First Tier 3 (Living Spectrum) device registered

## Far — v1.0.0 · *The Spectorium Stands*

**Theme:** The platform is stable, documented, and running. Ready for the world.

- [ ] All 31 trees represented by at least one Grove
- [ ] Full Sudoku Engine integration — valid board states across Groves
- [ ] Multi-Haven federation — Spectorium forks linked via NETWORK.md
- [ ] Grove versioning — device Groves maintain their own semantic versions
- [ ] Automated chain-of-custody report — exportable per device
- [ ] The Spectorium 1.0 release — stable standards, no breaking changes

## Concept Backlog

Ideas that belong in the Spectorium but haven't found their milestone yet:

- **The Splinter Index** — a searchable record of all Splinters found across the forest
- **Sparkle Protocol** — formalized rules for what constitutes a Sparkle and how it propagates
- **Grove Genealogy** — visualizing which Groves are descendants of which Havens
- **The 82nd Position** — what exists outside the Sudoku Engine's 9×9 grid?
- **Neurodivergent Grove Standard** — specialized guidance for Tier 3 Living Spectrum devices
- **The Oaken Ritual** — a formal process for a new Haven to acknowledge Turn 0
